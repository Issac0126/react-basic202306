
# React Project 시작하기

1. node.js 설치 (LTS 버전으로)
2. create-react-app 설치 (최초 한 번만 진행)
```
$npm install -g create-react-app
```

3. react 프로젝트 생성
```
$ npx create-react-app
```

4. react 프로젝트 실행
```
$cd 프로젝트폴더
$ npm start
```

- http://localhost:3000 에서 프론트엔드 서버 실행.

## git clone 시 주의사항
---
```
$ npm install
```
명령을 터미널에서 실행하여 라이브러리 설치가 필요하다.
